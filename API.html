Om te oefenen met hoe API's werken kregen we de opdracht om 2 verschillende API's te maken, de eerste is een weer API. De opdracht hiervoor was dat we van een plaats naar keuze het weer moesten laten zien op een webpagina doormiddel van een API, voor de plaats heb ik mijn geboorte dorp Reuver gekozen. Reflectie

Ik vind het werken met API's erg interessant. Doordat ik mijzelf nog meer ben gaan verdiepen in API's kwam ik er ook achter hoeveel website's er wel niet gebruik van maken. Voor het project dat ik met Evan en Ard doe maken we ook gebruik van een API om te kunnen communiceren met ons prototype. Voor de toekomst ga ik zeker nog eens kijken naar welke API's er allemaal zijn en wat er mogelijk mee is, graag zou ik zelf een websitepagina willen maken waarin ik gebruik maak van een API die gekoppeld is aan mijn persoonlijke CryptoWallet.
